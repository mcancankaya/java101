## Sum Of Digits
A program that finds the sum of the digits of the number entered by the user.
## Sum Of Digits Method
```java
while (tempNumber != 0) {
            total += tempNumber % 10;
            tempNumber /= 10;
        }
```

```
Enter a number :256

```
